# Overview
사내 프로젝트로 개발한 웹 어플리케이션을 서비스하기 위해 Ubuntu가 설치된 서버에 환경을 구성중입니다.
이 포스트는 docker를 기반으로 웹 서비스 환경을 구성하기 위한 절차를 정리하기 위해 작성되었습니다.
사용된 기술은 다를 수 있지만 docker를 이용하여 웹 서비스 환경을 구성하실 때 참조하시면 좋을 것 같습니다.
